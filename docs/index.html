Overview

simple_onnx is a production-ready wrapper for the ONNX Runtime C API enabling high-performance machine learning inference in Eiffel applications. It provides type-safe tensor manipulation, multi-format I/O, and Design by Contract verification without requiring Python or external runtime dependencies.

Part of the Simple Eiffel ecosystem.

Features

  • Direct C API Integration: Inline C externals call ONNX Runtime C library directly
  • Type-Safe Tensors: Support for float32, int32, int64, bool, float16 with full contract verification
  • Multi-Execution Providers: CPU (default), CUDA, TensorRT, CoreML backend selection
  • Immutable Results: XOR pattern ensures success or error, never both
  • SCOOP Compatible: Safe for concurrent/parallel execution
  • Void Safe: void_safety="all" enforced throughout
  • Design by Contract: 91+ contract clauses ensure correctness

Quick Start

Load a model and run inference: 

local
    onnx: SIMPLE_ONNX
    model: ONNX_MODEL
    session: ONNX_SESSION
    input_tensor: ONNX_TENSOR
    result: ONNX_RESULT
do
    create onnx.make
    model := onnx.load_model ("model.onnx")
    session := onnx.create_session (model)
    input_tensor := onnx.create_tensor_float32 (shape)
    result := session.execute (input_tensor)
    if result.is_success then
        process (result.output_tensor)
    end
end

See Quick API Reference for more examples.
